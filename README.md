# CourseRecommendBot
2023 LAT final project, in NTNU

## Describe
* 動機：
  提供大學生學習時的選課建議，通過chatBot的形式回應對學習內容topic有幫助的課程名稱
* ### Environment：
  - Python 3.10.11  
  - Node.js 9.5.1  
  - Windows 11 Home  
  - ...More in package-lock.json  

## Build
### Setup
    npm install
    pip install openai
### Run
    npm run start
***
### Attension
* 你需要自行修改key.txt，並放在目錄key資料夾下
